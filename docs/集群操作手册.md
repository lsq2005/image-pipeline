## 项目操作手册：启动与关闭

### 一、开机（阿里云控制台）

1. 登录阿里云 ECS 控制台
2. 将三台实例（master、worker1、worker2）**开机**
3. 确认弹性公网 IP（`121.43.161.9`）已绑定到 master

---

### 二、启动 Hadoop 集群

SSH 连接到 master（以 root 用户）：

```bash
ssh root@121.43.161.9
```

启动 HDFS 和 YARN：

```bash
start-dfs.sh
start-yarn.sh
```

验证集群状态：

```bash
jps                     # 应有 NameNode、SecondaryNameNode、ResourceManager
yarn node -list          # 确认两个 worker 都是 RUNNING
```

---

### 三、启动 GNOME 远程桌面（按需）

如果上次关机前关闭了 xrdp 服务，启动它：

```bash
sudo systemctl start xrdp
```

从 Windows 连接：按 `Win+R`，输入 `mstsc`，计算机名填 `121.43.161.9`，用户名 `root`。

**关闭桌面但不影响集群**：直接在远程桌面里"注销"，或执行 `sudo systemctl stop xrdp`。

---

### 四、启动 Jupyter Lab

在 master 终端（SSH 或桌面终端，root 用户）执行：

```bash
jupyter lab --ip=0.0.0.0 --port=8888 --no-browser --allow-root
```

从 Windows 本机访问（推荐用 SSH 隧道）：

在 Windows PowerShell 新开窗口：

```powershell
ssh -L 8888:localhost:8888 -L 8088:master:8088 -L 4040:master:4040 root@121.43.161.9
```

然后在浏览器访问：

```
http://localhost:8888/lab?token=你的token
```

---

### 五、关闭流程

#### 1. 停止 Spark 应用

在 Jupyter 里执行：

```python
spark.stop()
```

#### 2. 关闭 Jupyter Lab

在运行 Jupyter 的终端按 `Ctrl+C`，然后关闭终端窗口。

#### 3. 关闭 Hadoop 集群

SSH 到 master：

```bash
stop-yarn.sh
stop-dfs.sh
```

#### 4. 关闭远程桌面

注销桌面（推荐），或 `sudo systemctl stop xrdp`。

#### 5. 关机

在 ECS 控制台**停止**三台实例，选择 **"节省停机模式"**。

### 六、常用命令速查

| 操作 | 命令 |
|------|------|
| 启动 HDFS | `start-dfs.sh` |
| 启动 YARN | `start-yarn.sh` |
| 查看 Java 进程 | `jps` |
| 查看 YARN 节点 | `yarn node -list` |
| 查看 HDFS 状态 | `hdfs dfsadmin -report` |
| 启动 Jupyter | `jupyter lab --ip=0.0.0.0 --port=8888 --no-browser --allow-root` |
| 查看 PySpark 环境 | `pyspark --version` |
| 查看 HDFS 文件 | `hdfs dfs -ls /` |
| 上传文件到 HDFS | `hdfs dfs -put 本地文件 HDFS路径` |
